## [0.8.0](https://github.com/willcgage/free-dispatcher/compare/v0.7.0...v0.8.0) (2026-06-29)

First desktop release — Free Dispatcher now ships as an installable host app.

### Features

* Desktop app packaging for Windows, macOS, and Linux — double-click host, no terminal ([#32](https://github.com/willcgage/free-dispatcher/issues/32))
* In-app auto-update with the running version shown in the UI ([#52](https://github.com/willcgage/free-dispatcher/issues/52))
* Physical Emergency Stop via pluggable command-station adapters (JMRI/WiThrottle) ([#55](https://github.com/willcgage/free-dispatcher/issues/55))
* Standalone session archiving + a past-sessions management list ([#57](https://github.com/willcgage/free-dispatcher/issues/57))
* Module Repository integration — catalog sync and an in-app module picker ([#50](https://github.com/willcgage/free-dispatcher/issues/50))
* In-app Release Notes — an offline changelog viewer ([#66](https://github.com/willcgage/free-dispatcher/issues/66))

### Bug Fixes

## 0.7.0 - 2025-12-26
- Added Electron shell and IPC-based backend URL resolution for the React frontend.
- Bundled FastAPI backend via PyInstaller binary; default to per-user SQLite storage with Postgres override.
- Added Electron packaging config and scripts (`electron:build`, `backend:bundle`).
- Retired Docker-based workflow and removed LAN socat scripts; updated documentation accordingly.
- Ignored PyInstaller artifacts and slimmed packaged assets to include only bundled backend.
